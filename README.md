## How To Use It ?

下載過後，在電腦中找到檔案的位置

![](https://i.imgur.com/TLLqsbW.png)

點兩下打開，可能會有以下兩種情況 :

- 有出現對話框 :
  
  - 點擊 "是"

- 沒有出現對話框 :
  
  - 這是正常的不用管他

![](https://i.imgur.com/QiNTpiO.png)

如果以上步驟都正常執行，那你將會看到一個視窗跳出來

![](https://i.imgur.com/EYINxJ8.png)

視窗中有一個顯示欄位 `Status` :

- ON
  
  - 學生端的廣播程式為開啟狀態

- OFF
  
  - 學生端的廣播程式為關閉狀態

以及兩個按鈕 :

- KILL
  
  - 關閉學生端的廣播程式，老師不會再 ***"借 "*** 到你的畫面

- RESUME
  
  - 開啟學生端的廣播程式，讓老師可以 ***"借 "*** 到你的畫面
